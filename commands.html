ℹ️ 终端输入防踩坑指南：
  • 指令格式演示中的方括号 [] 仅作为参数占位符，实际在游戏中输入时，请直接写内容，不要输入方括号本身。
  • 当要输入的字符串（如网格名称、清单名等）中带有空格时，请使用半角英文双引号将其包裹，例如："My Ship"。
🚁 存取飞船船坞系统
想做什么指令说明 / 注意
保存飞船!网格 保存 [船名] [true/false]!hangar save / !h save不填名字默认存眼前的船；第二参数 true = 模糊匹配船名（有重名时用）
示例
!网格 保存 保存眼前看到的船（推荐）
!网格 保存 矿机 保存名字叫「矿机」的船
!网格 保存 矿机 true 保存名字含「矿机」的船（有重名时用）
取出飞船!网格 加载 [船名] [模糊] [原地] [完全原地]!hangar load / !h load在你面前生成，按与保存位置的距离收费；距离 <1km 免费
参数说明与费用
第二参数：模糊匹配 true/false
第三参数：是否原地（即保存时坐标）生成 true/false
第四参数：完全原地（无碰撞检测）true/false，可能K爆，后果自负；仅当第三参数为 true 时有效

费用 ≈ (估价费+规模费) × (距离倍率+重力倍率)；船厂/市场首次加载免费
原地取出!网格 原地加载 [船名] [模糊] [完全原地]!hangar loadlocal从保存时坐标原地加载；不需人在旁边可远程使用；完全原地参数 true = 无碰撞检测，⚠️ 原位有网格会K爆
示例
!网格 原地加载 矿机 普通原地加载
!网格 原地加载 矿机 false true 第一参数 false=精确匹配，第二参数 true=完全原地（危险）
查看船坞列表!网格 列表 [过滤词]!hangar list / !h list列出所有保存的船；可加关键词过滤
查询保存位置!网格 船坞位置 [船名]!hangar pos / !h pos查看该船保存时所在的坐标

📦 存取物资仓库系统
想做什么指令说明 / 注意
存入指定物资!仓库 存入 [物品] [数量]!storage store / !storage save不需对准箱子，自动从整个网格收集材料；重力 >0.5G 时按物品价值收费，需输入两次指令确认
存入所有物资!仓库 存入全部!storage storeall把船里所有东西存进仓库
只存矿石!仓库 存入全部矿物!storage storeores仅存矿石类，其他物品不动
取出指定物资!仓库 取出 [物品] [数量] [箱标记]!storage draw从仓库取出放进当前船箱子
[箱标记] 参数说明
不填：物品放入你眼前指向的箱子
填写标记字符串：在眼前网格的所有箱子里，找自定义数据中包含该字符串的箱子放入
示例：!仓库 取出 钢板 100 "零件箱"
箱子装不下时会尽量装满，剩余返还仓库
取出所有物资!仓库 取出全部 [箱标记]!storage drawall取出全部（不填箱标记时需对准一个箱子；箱标记用法同上）
取物资到背包!仓库 取出到玩家 [物品] [数量]!storage drawtoself / !storage drawp直接放到玩家背包身上
查看仓库内容!仓库 列表!storage list列出仓库所有物品和数量
按清单存入!仓库 存入清单 [清单名称]!storage storelist按预定义清单批量存入物品
按清单取出!仓库 取出清单 [清单名称] [箱标记]!storage drawlist按清单批量取出物品到飞船
绑定飞船!仓库 绑定!storage bind绑定后仓库材料不足时自动从此船物流补充
解绑飞船!仓库 解除绑定!storage unbind解除当前绑定关系
